# IKS-HACKTHON
NaadaTaranga is an AI-based Sanskrit chanting tool that generates rhythmic audio from text using chandas and TTS, combining traditional linguistic rules with modern technology.
# How to Run The Model
 Clone the repository
 Install dependencies: pip install -r requirements.txt
 Run the app in the terminal: python app.py
 Open in browser: http://127.0.0.1:5000/
